# Licensing

Solutions is available to all customers. However, using certain components within a solution may depend on whether [the relevant product](https://docs.uipath.com/solutions-management/automation-cloud/latest/user-guide/solution-components) is licensed.

For licensing information on each product, visit [Service licensing](https://docs.uipath.com/overview/other/latest/overview/service-licensing).

Additional resources:
- [Detailed information on licensing plans, license assignments, and monitoring](https://docs.uipath.com/automation-cloud/automation-cloud/latest/admin-guide/about-licensing)
- [General information on user and service licensing](https://docs.uipath.com/overview/other/latest/overview/licensing-levels)
