# Accessing the Solutions UI

> **Note:** Accessing  from Automation Ops now redirects you to Solutions in Orchestrator.

You can access the  user interface through Orchestrator by taking the following steps:

1. Navigate to your tenant context within Orchestrator.
2. Select the **Solutions** tab.
